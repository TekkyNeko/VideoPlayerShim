# VideoPlayerShim
This is a Video Player Shim to allow video players (Unity Video and AVPro) to work in PoligonVR CCK World Projects

## Installation
- Import the Unitypackage in Releases. (release soon-ish)
- When prompted to download AVPro, hit yes.
- Make sure you have launched PoligonVR at least once so PVR's yt-dlp is downloaded.
- You are done! Go into play mode with a video player and you should be able to use it.
